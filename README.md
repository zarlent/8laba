# Домашнее задание к работе 8

## Условие задачи
Вычислите `1ᴷ + 2ᴷ + … + Nᴷ`. Для заданного пользователем целого N и K.    
Чтобы избежать целочисленного переполнения, вычислять как вещественное.

## 1. Алгоритм и блок-схема

### Алгоритм
1. **Начало**
2. Ввести целое число `N,K`
3. Для `i` от 1 до `N` (включительно) выполнить:  
  - Вычислить `pow(i, K)` — i в степени K
  - Прибавить результат к `sum`
4. Вывести значение `sum`
5. **Конец**

## 2. Реализация программы

```
#include <stdio.h>
#include <locale.h>
#include <math.h>

int main() 
{
    setlocale(LC_CTYPE, "RUS");

    int N, K;
    double sum = 0.0;

    printf("Введите число N: ");
    scanf("%d", &N);

    printf("Введите число K: ");
    scanf("%d", &K);

    for (int i = 1; i <= N; i++) {
        sum += pow(i, K); 
    }

    printf("Результат вычислений: %.0lf\n", sum);

    return 0;
}

```
